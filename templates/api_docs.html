Prepared Structure Retrieval

Warhead Hunter example jobs can expose cleaned ligand-bound PDB files, ligand SDF files, SVG atom maps, SASA tables, and downloadable result bundles. This makes the API useful for researchers who want to collect prepared protein-ligand structures by target class, protein name, or completed job ID.

These are prepared, cleaned, job-derived outputs. They should be treated as available outputs from completed Warhead Hunter runs, not as a validated dataset.

  • Notebook-based structure inspection
  • Ligand-bound PDB collection
  • PROTAC linker/vector analysis
  • Comparison across protein families
  • Feeding cleaned structures into companion tools

Clean Result Bundles

Warhead Hunter jobs can include a curated public result bundle generated by 18_CleanJobDirNzip.py. This bundle is intended for downstream users who want cleaned/prepared PDB files, ligand SDFs, SVG atom maps, SASA tables, and final result summaries without repeated scripts or intermediate clutter.

If a curated public bundle is not yet present, the API falls back to the existing safe bundle behavior.

19. Python Example

        

          

            
          

          
import requests

BASES = [
    "{{ primary_api_base }}",
    "{{ secondary_api_base }}",
]

def choose_base():
    for base in BASES:
        try:
            r = requests.get(f"{base}/api/health", timeout=5)
            if r.ok:
                return base
        except requests.RequestException:
            pass
    raise RuntimeError("No Warhead Hunter API base is reachable")

base = choose_base()
examples = requests.get(f"{base}/api/examples").json()
print(examples)

job_id = "{{ preferred_example_job_id }}"
files = requests.get(f"{base}/api/examples/{job_id}/files?kind=pdb").json()
print(files)

        

      


      

        
20. JavaScript Example

        

          

            
          

          
const BASES = [
  "{{ primary_api_base }}",
  "{{ secondary_api_base }}"
];

async function chooseWarheadHunterBase() {
  for (const base of BASES) {
    try {
      const controller = new AbortController();
      const timer = setTimeout(() => controller.abort(), 5000);
      const res = await fetch(`${base}/api/health`, { signal: controller.signal });
      clearTimeout(timer);
      if (res.ok) return base;
    } catch (err) {}
  }
  throw new Error("No Warhead Hunter API base reachable");
}
