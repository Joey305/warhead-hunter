Workflow Story

API Workflow At A Glance

Submit jobs with target_name and search_query, monitor status, retrieve file manifests, download WAR_PDB structures, and automate downstream structure collection.

Input
Submit Job
POST target_name, search_query, and optional fasta_seq.
Status
Monitor
Poll active job status and browser monitor URLs.
Results
Manifest
Inspect key outputs, counts, and result paths.
Artifacts
Files
List WAR_PDB, SDF, SVG, table, HTML, and manifest files.
Download
ZIP Bundles
Download curated result bundles or WAR_PDB-only ZIP archives.
Scale
Batch API
Launch multiple jobs sequentially and monitor them as one batch.

Build Structure Libraries
