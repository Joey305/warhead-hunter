Core Capabilities

The Target Module acts as the structural intelligence loader for the entire degrader pipeline. Every ligand uploaded is normalized, aligned, and mapped into a unified computational format.

  • MCS Atom Mapping: 2D→3D deterministic atom correspondence.
  • SASA Extraction: Interface exposure & warhead positioning heuristics.
  • Scaffold Assignment: Generic + chemotype-level grouping.
  • Attachment Vector Computation: Warhead geometry constraints.
  • PDB/CIF Harmonization: Chain fixing, atom renumbering, residue normalization.
  • Packaged System Export: Ready for scoring, MD, docking, and PRosettaC.

Technical Architecture

The pipeline ensures every ligand + receptor pair is computed with complete reproducibility, regardless of user input format.

  • Metadata Harvester: SMILES, scaffolds, names, IDs, chemotype tags.
  • Structure Ingestion: Multi-format support (PDB, CIF, SDF, MOL2).
  • MCS Engine: RDKit-driven deterministic topology mapping.
  • Exposure Engine: SASA, warhead solvent vectors, rigid site classification.
  • CHEM AI Prep: Embedding ligand metadata for future ML models.

Scientific Foundation

Degrader modeling is sensitive to warhead geometry, linker orientation, and interface exposure. The module ensures these remain consistent through:

  • Attachment vectors and SASA heuristics
  • MCS atom anchoring for multi-target comparisons
  • Scaffold similarity + chemotype clustering
  • Interaction density benchmarking
  • Support for AF3 and PRosettaC structural constraints
