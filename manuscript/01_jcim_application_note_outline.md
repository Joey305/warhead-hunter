## Abstract Options

### Abstract Option 1

Warhead Hunter is a web-based software workflow for analyzing ligand-bound protein structures and mapping solvent exposure at the ligand-atom level. Starting from structure retrieval and sequence-guided filtering, the platform computes ligand atom solvent-accessible surface area, links exposure values to 2D and 3D ligand representations, and presents results through an interactive browser interface. Warhead Hunter is designed to support structure-guided prioritization of candidate sites for warhead installation, linker attachment, and ligand expansion.

### Abstract Option 2

Warhead Hunter supports atom-resolved interpretation of protein-bound ligands by integrating structure retrieval, ligand-focused preprocessing, solvent-accessible surface area analysis, and 2D/3D atom mapping in a browser-accessible workflow. The software generates interpretable exposure maps that can help identify candidate ligand atoms for derivatization in covalent design, linker attachment, and induced-proximity programs. The repository and interface are organized around reproducible job folders and result artifacts suitable for future API extension.

### Abstract Option 3

Warhead Hunter provides a web workflow for converting ligand-bound protein structures into atom-level solvent exposure maps. The platform combines structure acquisition, sequence-aware filtering, ligand preprocessing, SASA analysis, RDKit-based atom mapping, and interactive visualization to highlight potentially modifiable ligand atoms in structural context. The software is intended for structure-guided medicinal chemistry tasks such as attachment-vector inspection, linker-placement hypothesis generation, and PROTAC-oriented ligand evaluation.

## Manuscript Angle And Contribution Statement

### Recommended angle

Warhead Hunter should be positioned as a scientific software platform that transforms ligand-bound protein structures into interpretable atom-level solvent exposure maps for structure-guided ligand modification planning.

### Why this angle is strong

- It is directly supported by the code.
- It avoids unsupported claims about predictive accuracy or medicinal chemistry success rates.
- It fits the current interface, pipeline outputs, and companion-tool ecosystem.
- It is well aligned with JCIM software-note expectations for transparency and utility.

### Contribution statement draft

Warhead Hunter integrates structure retrieval, ligand-focused preprocessing, atom-level SASA analysis, RDKit-based atom mapping, and web visualization into a single workflow that helps users inspect candidate ligand atoms for derivatization in protein-bound structural context.
